BitBIRCH-Lean documentation#

Date: Oct 02, 2025 Version: 0.0 (dev)

Useful links: GitHub Repo | Issue Tracker | Miranda-Quintana Lab at GitHub

BitBIRCH-Lean is an open-source, high-throughput implementation of the BitBIRCH clustering algorithm designed for clustering very large molecular libraries. It is developed by the Miranda-Quintana Lab at the University of Florida.

The BitBIRCH-Lean documentation is a work in progress, expect many updates soon!
